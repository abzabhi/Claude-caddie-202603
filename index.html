About This App
Virtual Caddie is a personal bag management and course strategy tool. All Optimizations are done by Claude AI or some other AI and can vary based on model and model provider. Use at your own risk. All in browser calculations are deterministic — the same inputs always produce the same outputs. This tab documents every assumption baked into those calculations.
Handicap & Scoring
WHS Differential
Formula: (Gross Score − Course Rating) × 113 ÷ Slope Rating, rounded to one decimal. This matches the World Handicap System standard.
Handicap Index
Uses best differentials from round history per the WHS table: 1 round → best 1; 7–8 rounds → best 2; 9 rounds → best 3; 10–11 → best 4; 12–14 → best 5; 15–16 → best 6; 17–18 → best 7; 19–20 → best 8. Average of selected differentials × 0.96 (WHS playing conditions calculation factor). Maximum 20 rounds stored.
Playing Handicap
Formula: round(HCP Index × (Slope ÷ 113) + (Course Rating − Par)). Applies to the selected tee and course.
Stroke Allocation
Holes are sorted by their stroke index (handicap column, 1 = hardest). A player receives 1 stroke on any hole whose index ≤ Playing HCP. If Playing HCP > 18, a second stroke is allocated on holes whose index ≤ (Playing HCP − 18).
Club Distance & Implied Yardage
Session Averages
avgMin and avgMax are simple means of all logged session values for that club. Each session contributes equally regardless of date.
Yardage Type
Default is Total (carry + roll). Set a global preference in Profile, or override per club in its specs panel. The label shown on each club reflects whichever is in effect — the data itself is whatever you logged.
Implied Yardage (Confidence Scaling)
Adjusts the displayed range based on your confidence in the club. The adjustment is symmetrical — both ends of the range move by the same percentage:
  • Score 5 (Highest): range tightens by 5% each end
  • Score 4 (Baseline): no adjustment — raw avg used
  • Score 3: range widens by 5% each end
  • Score 1–2 (Low/Least): range widens by 10% each end
Gap suppression: if avgMax − avgMin is already tight (≤ 15 yds for short clubs; ≤ 20 yds for long clubs including Driver, Fairway Wood, and hybrids with loft ≤ 30° or number ≤ 6H), confidence scaling is not applied — the raw range is already precise enough. Toggle implied yardage display on/off in the Clubs tab.
Long vs Short Club Threshold
Driver, Fairway Wood, hybrids ≤ 6H or loft ≤ 30° → Long (20 yd gap threshold). All other irons, hybrids ≥ 7H or loft > 30°, Wedges, Chipper → Short (15 yd gap threshold).
Ball Flight & Dispersion
Sources
Spin RPM and launch angle benchmarks from TrackMan published amateur data. Lateral dispersion calibrated against GOLFTEC launch monitor study (7-iron: scratch ~25 yds width, 8 HCP ~30 yds, 13 HCP ~35 yds) and DECADE / Practical Golf (driver: scratch ~65–70 yds total width). Longitudinal miss distances from HackMotion research. Arccos GPS data used for cross-validation.
Handicap Tiers
Six tiers: 0–4 · 5–9 · 10–14 · 15–19 · 20–24 · 25+. If no handicap is set, tier 3 (15–19) is assumed.
Iron Classification
Irons are split into Long (2–4i, loft ≤ 27°), Mid (5–7i, 28–38°), and Short (8–9i, 39–46°) groups for dispersion lookup. Classification uses the loft° value in the identifier field first, then the iron number.
Lateral Dispersion
Expressed as a ± radius (half of total lateral width). Adjusted by bias multiplier: Straight ×1.00 · Draw ×0.95 · Fade ×0.95 · Hook ×1.15 · Slice ×1.20. Lateral offset applied to centreline: Draw −5 yds · Hook −15 yds · Fade +5 yds · Slice +15 yds (positive = right for right-handed player; mirrored for left-handed).
Longitudinal Dispersion
Short and long miss distances are tracked separately — players at all levels tend to miss shorter more than longer. These values inform forced-carry decisions in the caddie session.
Home & Favourite Course
Home course is set manually by toggling the button on any course card. Only one course can be home at a time. The home course always appears first in the course list and is shown in your profile tagline.

Favourite course is calculated automatically as the course name appearing most frequently in your round history. It updates in real time as you log rounds. If the favourite is the same as home, only the Home badge is shown.
Data & Privacy
All data is stored locally in your browser's localStorage. Nothing is sent to any server. Exporting a .txt file is the only way to back up or transfer your data between devices. Clearing browser data or using a different browser will reset the app — always export before clearing. The app version key (APP_VERSION) is used to detect stale data from older versions and will reset stored data on a major version change.
Flight Data Version
Current flight reference data version: 1. When this version is bumped in a future update, exported TXT files will carry the old version number in their FlightDataVersion header — this allows the caddie skill to detect and flag any mismatch between stored session data and the dispersion tables in use.
